# Security Policy

## Supported Versions

Only the latest version of the script is supported for security updates.

| Version | Supported |
| ------- | --------- |
| 3.x     | ✅         |
| 2.x     | ❌         |
| < 2.0   | ❌         |

## Reporting a Vulnerability

If you discover a security vulnerability, please do not open a public issue. Instead, report it by:

- Opening a private security advisory on GitHub (if available).
- Or contacting the maintainer via their profile.

We take security seriously, especially since this script modifies network protocols like SMBv1.
